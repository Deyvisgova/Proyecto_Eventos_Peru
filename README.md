# Proyecto Eventos Perú
Incluye el backend (Spring Boot) y el frontend (Angular) en un solo repositorio.

## Diagramas de arquitectura
Los siguientes recursos visuales ayudan a explicar conceptos clave de la solución:

- **Implementación de patrones**: muestra cómo Factory Method, Strategy y Observer/Mediator colaboran en los servicios de eventos. ![Implementación de patrones](docs/imagenes/implementacion_patrones.svg)
- **Arquitectura por capas**: ilustra la separación entre presentación, aplicación/servicios, dominio e infraestructura. ![Arquitectura por capas](docs/imagenes/arquitectura_por_capas.svg)

Para ver las imágenes directamente y en mayor tamaño dentro del repositorio, consulta `docs/imagenes/README.md`.
